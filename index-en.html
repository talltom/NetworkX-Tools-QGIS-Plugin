Overview

The NetworkX Tools Quantum GIS Plugin uses the NetworkX package to create and analyse networks created from spatial vector data. Currently, only shapefile data are supported for reading and writing networks.

NetworkX Tools is developed by Tom Holderness at Newcastle University and is released under and open-source BSD style license. Developers can access source code at the github page:

Installation

Install the plugin by adding the repository link to Quantum GIS (Plugins -> Fetch Python Plugins -> Repositories -> Add repository). Once the repository is added click on the "NetworkX Tools" package in the "plugins" tab to install the package. One installation is complete you should see "NetworkX Tools" available in the "Plugins" menu.

QGIS repository: http://www.students.ncl.ac.uk/tom.holderness/qgis_repo.xml

To run the plugin you will also need the NetworkX package available from: http://networkx.lanl.gov/

Building network files

NetworkX creates networks by specifying nodes and interconnecting edges. To create NetworkX network shapefiles:
  1. Add a shapefile line layer to QGIS.
  2. Open the "Build Network" tool: (Plugins -> NetworkX Tools -> Build Network).
  3. Select the loaded layer in the available layers drop-down.
  4. Select an output folder.
  5. Check the "Add new files to QGIS" tick box.
When you click "OK" NetworkX will read your shapefile line layer and create a network of nodes and interconnecting edges as defined by the geography and order of line creation in the shapefile. It will then create two new shapefile layers "nodes" and "edges" which represent the network. Attributes in the original shapefile are copied to the "edges" layer.

Shortest Path analysis

The shortest path analysis tool calculates the shortest weighted or un-weighted distance between two nodes in a network. Note that the network should be separated into edge and node layers (e.g. use the "Build Network Tool" to create the required files. To perform create shortest path analysis:
  1. Open the "Shortest Path" tool (Plugins -> NetworkX Tools -> Shortest Path)
  2. Select the desired algorithm
  3. Select the node layer
  4. Select the corresponding edge layer
  5. Specify weighting if required (attributes from edge layer)
  6. Click the Source node button to select path origin from node layer
  7. Click the Target node button to select path destination from the node layer
  8. Specify and output folder
  9. Click OK
When you click OK NetworkX will determine the shortest route over the network between the specified points and will create two new shapefiles corresponding to the edges and the nodes for the shortest path.
